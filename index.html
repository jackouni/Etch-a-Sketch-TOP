WELCOME TO HEAVENS-SKETCH

This game was designed by Jack Sebben as a project for the free, open-source, self-taught web development curriculum, The Odin Project. If you want to see more of Jack's work and follow his journey as a self-taught web dev - social links can be found below.

There are 3 modes and one reset button to select from. We'll start by explaining the modes from the top button to the bottom.

Color Mode: This will allow you to pick a solid colour to use on the etch-a-sketch. By clicking this mode you will bring up a colour pallette where you can choose what colour you would like to use. This mode will fill your selected colour to each pixel on the etch-a-sketch that you move over.

Rainbow mode: Will fill a random colour to each pixel on the etch-a-sketch that you move over.

Eraser mode: Will remove coloured pixels from your etch-a-sketch as you move over them.

Reset: This button reset the etch-a-sketch, giving you a clean canvas to work on.

The etch-a-sketch is where you create your drawings. By selecting a mode and moving your cursor over the pixels on the etch-a-sketch, you will activate/use the mode that is currently active. Below the etch-a-sketch is a slider where you can adjust the etch-a-sketch's pixel density. By default the canvas is 16x16 pixels. By moving the slider left or right you can decrease or increase the number of square pixels in the etch-a-sketch.

A higher number = finer strokes
while a lower number = more blocky

** HAVE CAUTION: whenever you adjust the pixel density, the etch-a-sketch will reset, erasing whatever was on it to start fresh.

Info/Help: Located in the upper left, this icon will show the info modal when clicked. If you ever need help using the app, or want to see accredations for the site, you can visit this info modal.

Sound on/off: Located in the upper right, the speaker icon that will toggle the music to be on or off when clicked on.
